# Bug Tracker Frontend

React + Vite frontend for Bug Tracker Pro.

## Environment

Create a `.env` file from `.env.example` and set:

- `VITE_API_URL`
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Local Development

```bash
npm install
npm run dev
```

Default local URL: `http://localhost:3002`
